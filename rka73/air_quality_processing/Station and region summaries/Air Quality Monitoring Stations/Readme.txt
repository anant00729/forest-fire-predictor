This folder contains the following subdirectories
  and all these subdirectories have the CSV files that have been downloaded from:
    https://catalogue.data.gov.bc.ca/dataset/01867404-ba2a-470e-94b7-0604607cfa30

  * 1980-2008
    - This contains the following file that has the data regarding the
        geo-coordinates (lat and long) of the air quality monitoring stations of BC from 1980 - 2008.
        - bc_air_monitoring_stations.csv

  * 2009-LatestVerified
    - This contains the following file that has the data regarding the
        geo-coordinates (lat and long) of the air quality monitoring stations of BC from 2009 - 2020:
        - bc_air_monitoring_stations.csv
